open the console to see the functionality of the code

This program defines a function called reverseLettersInWords() which takes in a string called sentence. It splits the sentence into an array of words using the split() method, then uses the map() method to reverse the letters in each word and join them back to form a string, finally it joins the reversed words with a space and return. The output of the program will be "sihT si a tset ecnetnes" from "This is a test sentence".
